# 08 — CONFLICTS, GAPS, AND DECISIONS

## 1. Conflicts requiring resolution

### Live catalog vs locked brand

- Live patriotic products conflict with the locked “no patriotic core” rule.
- Decide whether these are legacy, seasonal, test, or permanent.
- Recommended: keep patriotic only as a controlled seasonal lane outside the brand center.

### Quiet heritage vs permanent B2G1

- B2G1 supports AOV but may train customers to wait and weaken premium perception.
- Recommended test:
  - Control: B2G1.
  - Variant A: Buy 2, unlock hat.
  - Variant B: pursuit bundle with 10–15% savings.
  - Variant C: threshold gift at a profitable AOV.

### Product claims vs competitive benchmarks

- GritFell uses imitation linen.
- Free Fly, Duck Camp, AFTCO, Huk, BURLEBO, and SITKA have stronger technical/fabric proof.
- Do not position GritFell as superior technical gear.
- Win on identity, art, story, gifting, fit range, value, and collection systems.

### Earlier broad creative framework vs v0.4 filter

Some older reports included Bird Nerd, Reel Cool Dad, cottagecore, coastal, Western, and affirmation slogans inside a broad GritFell universe. The later v0.4 system routes many of these outside core GritFell. The agent must treat v0.4 as the current authority.

- Current unit economics for every category.

- Product-level conversion rate.
- Product-level AOV and attach rate.
- Repeat purchase by category.
- Meta creative performance.
- Search-query conversion.
- Customer review themes.
- Return reasons and size complaints.
- Actual bestsellers by revenue.
- Geographic sales concentration.
- Production defect rate.
- Delivery-time distribution.
- Competitive paid-ad frequency and spend.

The agent must request or connect these before making budget decisions.

## 4. Decisions recommended now

1. Lock GritFell Heritage as the core cartridge.
2. Treat patriotic and Western/Texana as separate experimental lanes.
3. Prioritize waterfowl-in-flight, dog partnership, species accuracy, and ritual collections.
4. Build the first cross-product system rather than another random SKU.
5. Test offer architecture against B2G1.
6. Improve product proof: fabric close-up, fit, size range, reviews, shipping.
7. Build a weekly official-source competitor snapshot.
8. Connect internal store/ads/email data before projecting revenue.

## 6. Human approval gates

Human approval is required before:

- Adding a new brand cartridge.
- Changing core pricing.
- Changing the permanent offer.
- Claiming technical performance.
- Launching a cause/disaster product.
- Using a competitor-derived visual too closely.
- Spending paid-media budget.
- Converting an internal hypothesis into a factual brief.
